# Student Grade Manager

## Overview
A Java console application built for CSE 310 Applied Programming Module 2 
at BYU-Idaho. This program allows users to add students, enter scores, 
calculate averages, assign letter grades, and save all records to a file 
so data persists between sessions.

## Demo Video
[Watch the demo on YouTube](https://youtu.be/bbQ1hFlma6g)

## Features
- Add multiple students by name
- Enter and validate scores between 0 and 100
- Automatically calculate each student's average score
- Assign letter grades based on average (A, B, C, D, F)
- View full student records with all scores
- View a formatted grade report summary table
- Save records to grades.txt for persistent storage
- Automatically load saved records when the program starts

## Java Concepts Demonstrated
- Classes and Objects (Student inner class)
- ArrayList from the Java Collections Framework
- Loops (for loop, while loop, for-each loop)
- Conditionals (if-else chain, switch statement)
- Methods with parameters and return values
- File I/O (reading and writing with BufferedReader and PrintWriter)
- Exception handling with try-catch blocks
